Pemrograman Pemula

Deskripsi Kursus

Masuki dunia coding dari nol. Kursus ini akan membimbing Anda memahami logika pemrograman dan membuat aplikasi web sederhana pertama Anda menggunakan HTML, CSS, dan JavaScript, tiga pilar utama pengembangan web modern.

Materi Kursus (10 Modul)

  • Modul 1: Pengenalan Web & Cara Kerja Internet
  • Modul 2: Struktur Halaman dengan HTML5
  • Modul 3: HTML Semantik & Formulir
  • Modul 4: Dasar-dasar Styling dengan CSS3
  • Modul 5: Layouting (Flexbox & Grid)
  • Modul 6: CSS Responsif (Mobile-First)
  • Modul 7: Pengenalan Logika & JavaScript
  • Modul 8: Manipulasi DOM dengan JavaScript
  • Modul 9: Bekerja dengan API (Dasar)
  • Modul 10: Proyek Akhir (Membuat Website Portofolio)
